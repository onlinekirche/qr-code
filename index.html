QR-Code Generator

QR-Codes können ein nützliches Hilfsmittel sein.
In diesem Beitrag ist ein Anwendungsbeispiel erklärt.

Zum Erzeugen der Codes stellt die OnlineKirche hier ein Hilfsmittel zur Verfügung.
Warum?
Es gibt zwar viele QR-Code-Generatoren im Netz. Es ist aber nicht immer ersichtlich, ob die eingegebenen Daten in irgendeiner Form weiterverarbeitet werden.
Wir nutzen hier die OpenSource Bibliothek von Nayuki.
Der Code kann auf dieser Projektseite überprüft werden.

Noch zwei Schritte zum QR-Code

1. Den gewünschten Text in das Textfeld unten eingeben
2. Mit der rechten Maustaste den erzeugten Code als Bild-Datei herunterladen
